CSS Basics

Introduction to CSS

CSS (Cascading Style Sheets) is the code that styles web content. CSS basics walks through what you need to get started. We'll answer questions like: How do I make text red? How do I make content display at a certain location in the layout? How do I decorate my webpage with background images and colors?

What is CSS?

CSS is not a programming language; it's a style sheet language used to selectively style HTML elements.

p {
  color: red;
}

Save the CSS in a file named style.css and link it in your HTML document to apply these styles.

Anatomy of a CSS Ruleset

Each CSS ruleset includes a selector, properties, and values. Here is an example:

p {
  color: red;
  width: 500px;
  border: 1px solid black;
}

CSS: All About Boxes

Most HTML elements can be considered boxes with padding, borders, and margins. You can style these boxes with CSS properties like width, background-color, padding, border, and margin.

Centering an Image

Using display: block; and margin: 0 auto;, you can center images and other inline elements:

img {
  display: block;
  margin: 0 auto;
}
