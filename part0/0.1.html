HTML Basics Summary

What is HTML?

HTML (HyperText Markup Language) is a markup language used to structure content on web pages. It consists of elements that wrap content to define its function and appearance.

HTML Elements

Elements include:

  • Opening Tag: Begins the element (e.g., <p> for paragraphs).
  • Closing Tag: Ends the element (e.g., </p>).
  • Content: The actual text or media within the tags.

Attributes

Attributes provide additional information about elements. Example:

<p class="editor-note">Example text</p>

In this example, class="editor-note" is an attribute.

Nesting Elements

Elements can be nested inside each other. For example:

<p>My cat is <strong>very</strong> grumpy.</p>

HTML Document Structure

An HTML document has the following basic structure:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Page Title</title>
  </head>
  <body>
    Page content here.
  </body>
</html>

Text Markup

  • Headings: Ranges from <h1> to <h6> to denote importance.
  • Paragraphs: Use <p> for blocks of text.
  • Lists:
    • <ul> for unordered lists.
    • <ol> for ordered lists.
    • <li> for list items.
  • Links: The <a> element creates hyperlinks. Example: <a href="https://www.mozilla.org/en-US/about/manifesto/">Mozilla Manifesto</a>
